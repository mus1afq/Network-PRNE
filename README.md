# Network-PRNE
# Programming for Network Engineers 

# Overview   
The project automates network device configuration using SSH and Telnet connections, compares configurations, and performs remote setup tasks.


# Tasks  
- Delivery 1: Connect via SSH/Telnet, change hostname, save running config.  
- Delivery 2: Compare configs (running vs startup/local or security compliance).  
- Delivery 3: Configure interfaces, routing, ACLs, or IPSec.  
- Delivery 4: Present work with slides and testing results.


## Requirements  
- Language: Python 3.10+  
- Libraries: Netmiko  
- Style: PEP8, well-commented  


# Optional Enhancements  
Add features like a Flask web UI, multithreading, or database storage for extra credit.
